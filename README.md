# 基于SpringBoot的图书馆座位预约系统+小程序+论文参考示例

#### 介绍
技术栈+工具：SpingBoot + RabbitMQ + MySQL5.7 + Maven + IDEA2022 + 微信开发工具

系统功能：图书分类、图书归还、图书删除、图书在线阅读（需授权）、图书借阅、座位预约、预约签到、预约签退等
